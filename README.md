# LeiloesTDSat

## Nome do projeto
LeiloesTDSat

## Explicação sobre o projeto
Este projeto é um sistema de leilões desenvolvido em Java. O sistema permite cadastrar produtos, listar os produtos cadastrados e registrar a venda de itens. O objetivo é auxiliar no controle de produtos em uma casa de leilões.

## Tecnologias utilizadas
- Java
- MySQL
- Git
- GitHub
